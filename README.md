# Tetris-Game

A basic tetris game following Ania Kubow's course on Udemy. I will be making a customized version in a seperate repo.

The game covers the following JS methods:
- addEventListener()
- querySelector()
- querySelectorAll()
- keyCode()
- Math.floor()
- Math.random()
- length
- forEach()
- splice()
- clearInterval()
- setInterval()
- some()
- innerHTML()
- every()
- add()
- remove()
- contains()
